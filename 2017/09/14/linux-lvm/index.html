LVM操作记录

发表于 2017-09-14 | 分类于 linux

LVM映射

初始状态

df -h

fdisk -l

安装lvm2,xfs

硬盘分区

parted /dev/sdb

parted /dev/sdc

此时状态fdisk -l

创建PV

pvcreate /dev/sdb1

pvcreate /dev/sdc1

此时状态pvdisplay

创建VG

初始vgcreate vg61 /dev/sdb1

追加vgextend vg61 /dev/sdc1

此时状态vgdisplay

创建LV

lvcreate --name aidata -L 2T vg61

此时状态lvdisplay

格式化mkfs.xfs /dev/vg61/aidata

挂载

mkdir /aidata

mount /dev/vg61/aidata /aidata/

此时状态df -h

扩展LV

lvextend -L +18T /dev/vg61/aidata

此时状态lvdisplay

xfs_growfs xfs_growfs /aidata/

其他类型resize2fs /dev/vg/123

此时状态df -h
