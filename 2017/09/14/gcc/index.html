发表于 2017-09-14 | 分类于 c++

C++ 编程中相关文件后缀

  • 静态库 (archive) .a
  • C++源代码（需要编译预处理）.C .c .cc .cp .cpp .cxx .c++
  • C或者C++源代码头文件 .h
  • C++源代码（不需编译预处理） .ii
  • 对象文件 .o
  • 汇编语言代码 .s
  • 动态库 .so

静态库是编译器生成的一系列对象文件的集合。
