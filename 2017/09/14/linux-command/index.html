查看指定进程详细信息

查看指定进程详细信息的最后一行

  • mapped 表示该进程映射的虚拟地址空间大小，即该进程预先分配的虚拟内存大小
  • writeable/private 表示进程所占用的私有地址空间大小，即该进程实际使用的内存大小
  • shared 表示进程和其他进程共享的内存大小
